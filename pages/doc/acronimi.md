## Specifiche

Utilizzare un browser per effettuare delle ricerche e scoprire cosa significano i seguenti acronimi:


* PC
* CPU
* GPU
* RAM
* ROM
* HDD
* SSD
* CD

Per ogni acronimo, descrivere con un massimo di due righe il componente e/o il funzionamento.

::right::

## Consegna

Scrivere un documento di testo, utilizzando Google Documenti, in cui scrivere l'elenco.

* DVD
* USB
* HDMI
* VGA
* LAN
* WAN
* HTML
* HTTP
* URL

Il nome del file deve essere "tuoCognome - Esercizio acronimi"
